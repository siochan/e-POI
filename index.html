e-POI

Collecting POI can be difficult. Let us help you navigate it like a pro!

e-POI provides a streamlined process of collecting and updating point of interest (POI) data.

E-POI will incentivize the users to collect accurate data by a play-to-earn approach and cryptocurrency-based rewards. The project will use blockchain to secure immutable POI data. The process for the user will involve - walking to the POI, 3D scanning the location, data gets verified and converts real-world data into a digital copy, and finally, users receive rewards.

-Decentralizes POI data collection

-Improves POI data quality

-Incentivizes users

-Captures multi-layered POI Data
